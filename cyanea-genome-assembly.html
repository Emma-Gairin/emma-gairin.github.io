Chrysiptera cyanea - Genome Assembly Pipeline

Chrysiptera cyanea is a common damselfish living in Indo-Pacific waters. To obtain its genome, its DNA was sequenced using PacBio HiFi, and the following pipeline was then used starting from the raw reads. The resulting genome and annotation are published at DOI: 10.46471/gigabyte.144.

1. Quality Control (QC) of Raw Reads

Before starting assembly, we perform quality control using tools like FastQC to evaluate the raw read quality.

2. Trimming

Low-quality bases and adapter sequences are removed using Trimmomatic to ensure clean input data for assembly.

3. Assembly

The genome assembly is performed using SPAdes or Canu, depending on the type of sequencing (short or long reads).

4. Post-Assembly QC

After assembly, we check for completeness and contamination using tools like BUSCO to assess genome assembly quality.

5. Annotation

Gene prediction and functional annotation are performed using tools like MAKER and Augustus.

For more details on the specific scripts used, see our Scripts Page.
